# ES6_basic

This project is part of the software engineering curriculum, focusing on the implementation of modern JavaScript (ES6 - ECMAScript 2015) features. The goal is to understand and apply new syntax and concepts introduced in ES6, such as block-scoped variables, arrow functions, and template literals.

## Requirements

*   **OS:** Ubuntu 20.04 LTS
*   **Runtime:** Node.js 20.x.x
*   **Package Manager:** npm 9.x.x
*   **Linting:** ESLint
*   **Testing:** Jest

## Setup

1.  **Install dependencies:**
    ```bash
    npm install
    ```
