# ES6 Classes

This project is part of the Holberton School curriculum, focusing on JavaScript ES6 classes, syntax, and object-oriented programming concepts.

## Requirements

* Allowed editors: `vi`, `vim`, `emacs`, `Visual Studio Code`
* All your files should be interpreted/compiled on Ubuntu 18.04 LTS using `node 12.11.x` and `npm 6.14.x`
* All your files should end with a new line
* The first line of all your files should be exactly `// eslint-disable-line` (or configured appropriately)
* A `README.md` file, at the root of the folder of the project, is mandatory
* Your code should use the `js` extension

## Tasks

* **0. ClassRoom** (`0-classroom.js`)
  * Implement a class named `ClassRoom` that accepts one attribute named `maxStudentsSize` (Number) assigned to `_maxStudentsSize`.
